## Overview
This lab focuses on implementing **multi-screen navigation** in a React Native application using **React Navigation**. The objective is to understand how users move between screens and how navigation state is managed within a mobile application.

## Project Description
The project is a **multi-screen React Native application** that demonstrates stack and tab navigation, as well as passing parameters between screens to support dynamic navigation behavior.

## Key Concepts
- React Navigation  
- Stack Navigator  
- Tab Navigator  
- Navigation parameters  
- Conditional navigation  

## Features Implemented
- Stack navigation between multiple screens  
- Bottom tab navigation for main sections  
- Passing parameters between screens  
- Dynamic screen titles using route parameters  
- Custom navigation headers
